# Tutor Price Prediction (Kaggle Regression)

## Описание проекта
В рамках проекта решается задача регрессии: **предсказание средней стоимости услуг репетитора за минуту (руб/мин)** на основе информации о преподавателе (предмет, рейтинг, отзывы, образование, опыт, текстовые описания и др.).

## Данные
Используются файлы соревнования:
- `train.xlsx` — обучающая выборка (есть целевая переменная `mean_price`)
- `test.xlsx` — тестовая выборка (без `mean_price`)
- `sample_submission.csv` — пример формата сабмишна

## Используемые признаки
- **Числовые:** `tutor_rating`, `tutor_reviews`, дополнительные engineered признаки  
- **Категориальные:** предмет, статус, опыт, степени и др. (One-Hot Encoding)  
- **Текстовые:** `description`, `experience_desc`, `categories`, `tutor_head_tags` (TF-IDF)

## Предобработка
Реализована через `Pipeline` и `ColumnTransformer`:
- заполнение пропусков (`SimpleImputer`)
- кодирование категорий (`OneHotEncoder`)
- векторизация текста (`TfidfVectorizer`)

## Модель
Использована **Ridge Regression** (линейная регрессия с L2-регуляризацией).  
Подбор параметра `alpha` выполнен через `GridSearchCV`.

## Оценка качества
Метрика соревнования: **RMSE**.  
Лучший результат на Kaggle: **52.60081 (26 место)**.

## Результат
Формируется файл `submission.csv` в формате:
- `index` — индекс строки тестового набора  
- `mean_price` — предсказанная стоимость (руб/мин)
